# PDF Editor

A lightweight desktop PDF editor application for basic PDF manipulation tasks.

## Overview

PDF Editor is a simple yet powerful tool that allows you to perform common PDF operations without the need for complex or expensive software. Whether you need to split a large PDF, convert documents to images, create PDFs from images, or combine multiple PDFs into one, this application has you covered.

## Usage

### Cutting/Splitting PDFs
1. Open the PDF file you want to split
2. Select the pages or page ranges to extract
3. Save the new PDF file

### Converting PDF to Images
1. Load the PDF file
2. Choose the output image format (PNG, JPG, etc.)
3. Select pages to convert
4. Save images to your desired location

### Converting Images to PDF
1. Select one or more image files
2. Arrange them in the desired order (if multiple)
3. Convert and save as a PDF file

### Merging PDFs
1. Add multiple PDF files to the merge list
2. Arrange them in the desired order
3. Merge and save as a single PDF file
