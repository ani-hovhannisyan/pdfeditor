# PDF Editor

A lightweight desktop PDF editor application for basic PDF editing tasks.

## Overview

PDF Editor is a simple tool that allows you to perform common PDF operations. Usefull to split a large PDF, convert documents to images, create PDFs from images, or combine multiple PDFs into one.

## Features

### PDF Cut/Split
Split large PDF files into smaller documents.

### PDF to Image Conversion
Convert PDF pages into image files (PNG, JPG).

### Image to PDF Conversion
Transform image files into PDF documents.

### Combine PDFs
Combine multiple PDF files into a single document. Maintain the order and formatting while creating a consolidated file from separate PDFs.

## Usage

### Cutting/Splitting PDFs
1. Choose the PDF file you want to split
2. Select the pages to extract
3. Save the new PDF file

### Converting PDF to Images
1. Load the PDF file
2. Choose the output image format (PNG, JPG, etc.)
3. Select pages to convert
4. Save images to your desired location

### Converting Images to PDF
1. Select one or more image files
2. Arrange them in the desired order (if multiple)
3. Convert and save as a PDF file

### Combine PDFs
1. Add multiple PDF files together
2. Arrange them in the desired order
3. Combine and save as a single PDF file
